# TP1 – RNN : Génération de musique

Ce TP explore l'utilisation des **Réseaux de Neurones Récurrents (RNN)** pour la **génération automatique de musique au format ABC**.  
Il comprend :  
- Prétraitement de données musicales textuelles.  
- Conception et entraînement d’un modèle RNN avec PyTorch.  
- Génération de nouvelles séquences musicales.
